As practice with single-page applications made with React, this app is a clone of the YouTube video player and allows users to instant-search for and watch videos.

<h1>Technologies Used </h1>
<ul> 
  <li>Node.js </li>
  <li> Bootstrap </li>
  <li>React </li>
  <li>lodash </li>
  <li>Youtube API </li>
</ul>
